# Ableton Stuff

|Name|Description|
|----|-----------|
| `ableton-themes-installer.sh` | Bash script. Drop this wherever you keep your downloaded Ableton themes (outside of the app folder), it'll copy them back to the Themes folder after an Ableton update wipes them out. Open Terminal, navigate to your themes folder and run with `bash ableton-themes-installer.sh`. Open the script in a text editor for more info if you hit permissions problems on MacOS 26. |
