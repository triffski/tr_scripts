# Scripts 'n stuff

Somewhere to store random scripts and whatnot. Everything was made for my own use and is provided as-is, without warranty yada yada... Read the docstrings for more info, back up your files first and do plenty of test runs before deploying fully. 

|  Name  |  Description  |
|--------|-----------|
|  `ableton`  | Quick reinstallation of Ableton themes from a master folder, if they're wiped out by a reinstall |
|  `album-artwork`  | Scan a music library to identify files with no artwork, pomote images found in album folders to album art and extract art from music files to a sidecar.  |
|  `exif`  | Scripts aimed at harvesting metadata from downloaded Facebook and Instagram archives, and transferring it to Immich. Also contains scripts to aid with Immich tasks that would be very (very) time consuming if done manually. |
|  `geo`  | Extract data from Google Maps for use with Darawich, OsmAnd etc |
