# Scripts 'n stuff

Somewhere to store random scripts and whatnot. Everything was made for my own use and is provided as-is, without warranty. 

Read the docstrings for more info, back up your files first and don't blame me if you didn't and something goes wrong! :)

| Name | Description |
|------|-------------|
| `ableton` | Quick reinstallation of Ableton themes from a master folder if they're wiped out by a reinstall. |
| `album-artwork` | Scan a music library to identify files with no artwork, promote images found in album folders to album art, and extract art from music files to a sidecar. |
| `exif` | Harvest metadata from downloaded Facebook and Instagram archives and transfer it to Immich. Also contains scripts to aid with Immich tasks that would be very tedious otherwise. |
| `geo` | Extract data from a Google Maps 'Takeout' for use with OsmAnd, Organic Maps etc. |
